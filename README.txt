COLOR PICKER by Tomer Shmul

# Summary

Use a pointer to select your color 
and have it be displayed on your screen!

# Program Instructions 

From the starting screen, press ENTER to start 

In the color palette screen: 
- use the 4 ARROW KEYS to move the pointer around
- press BACKSPACE to return to the starting screen 
- OR press ENTER to select the color that the pointer is at

On the chosen color screen:
- see the color you chosen
- press BACKSPACE to return to the starting screen


Features:
[X] DMA Images (startScreen, palette, cursor)
[X] Button input 
[X] 2D Movement
[X] Object Collision (getting the color the pointer is on)
[X] Text progression (second counter)
[X] No tearing
